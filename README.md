# LeaderCal — Marketing Site

Scheduling built for leaders. Part of the LeaderHQ suite.

## About

LeaderCal is the calendar and appointment-scheduling product in the LeaderHQ suite — sibling to LeaderLeads. Booking pages, team round-robin, executive event types, and calendar sync, wired into the tools modern leadership teams already use.

## Tech

Static HTML/CSS/JS. Zero build, no framework, no bundler. Hosted on Vercel edge.

## Local dev

```bash
npx serve .
```

Or just open `index.html` in your browser.

## Deploy

Auto-deploys to Vercel on push to `main`.

Production: https://cal.leaderhq.io
